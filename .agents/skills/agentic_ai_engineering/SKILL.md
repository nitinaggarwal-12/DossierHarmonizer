# 🤖 Agentic AI Project Engineering Standards

This skill provides unified development guidelines, state management locks, automated validation protocols, and pre-project planning checklists for all web engineering and generative AI projects.

## How to Apply:
1. **At the start of a task or project:** Read the unified reference document located in the references subdirectory: [guide.md](file://./references/guide.md).
2. **Pre-Project Planning:** Execute the Pre-Project Research & Planning Protocol (Section 10 of the reference guide) before writing or editing any code files.
3. **Database Caching & Security:** Standardize on Node-native SQLite caching and local data sanitization patterns when interacting with external APIs.
4. **Validation Reports:** Ingress test validation details to automate markdown qualification certificates.
